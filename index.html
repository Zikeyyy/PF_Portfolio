Featured Project · Azure ML & Automation

Credit Card Approval — End-to-End Azure ML Pipeline

Built a fully automated credit card approval pipeline on Azure. A logistic regression model (trained on a Kaggle dataset with SMOTEN oversampling, WOE/IV feature engineering, and joblib serialisation) was deployed to an Azure Function App. An 8-step Azure Logic App orchestrates the entire workflow — from receiving a PDF application via HTTP trigger, extracting structured data using Azure Document Intelligence, constructing the ML input payload, calling the prediction endpoint, and returning a HIGH/LOW priority classification. Achieved 81% accuracy with consistent end-to-end execution times of 13–14 seconds.
